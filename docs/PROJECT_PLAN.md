**프로젝트명:** K-Player Tracker
**작성일:** 2026. 02. 03
**버전:** 0.7.0
**MVP 타겟 선수:** 이강인
**서비스 비전:** 축구를 넘어 모든 해외파 한국 선수를 아우르는 통합 팬덤 플랫폼

---

## 1. 서비스 개요 (Overview)

### 1.1 서비스 정의
해외 리그에서 활약하는 한국 선수의 경기 일정, 스탯, 뉴스, 커뮤니티를 한눈에 제공하는 **'선수 중심 팬덤 플랫폼'**

### 1.2 배경 및 문제 정의
| 문제점 | 해결 방안 |
|--------|----------|
| 기존 스코어 앱은 정보가 파편화됨 | 선수 중심의 통합 대시보드 제공 |
| 도박/배팅 위주의 앱 환경 | 순수 팬을 위한 클린한 플랫폼 |
| 단순 숫자 나열식 정보 | 입체적 인포그래픽 데이터 시각화 |
| 커뮤니티 접근성 부족 | 원클릭 Talk 기능으로 즉각 소통 |
| 여러 선수 팔로우 불편 | 즐겨찾기 기반 멀티 선수 관리 |
| 비개발자의 콘텐츠 관리 어려움 | 자동화 파이프라인 + 승인 시스템 |

### 1.3 핵심 가치 (Core Value)
> **"내 선수의 모든 것을, 한 화면에서, 아름답게"**

### 1.4 USP (Unique Selling Point)
| 경쟁사 | 우리의 차별점 |
|--------|--------------|
| FotMob - 팀/리그 중심 | **선수 중심** 대시보드 |
| OneFootball - 정보 나열식 | **입체적 인포그래픽** |
| SofaScore - 데이터만 제공 | **커뮤니티 Talk 연계** |
| 기타 앱 - 도박 광고 범람 | **클린 플랫폼** |
| 기타 앱 - 경기날 변화 없음 | **매치데이 모드** |

## 2. 유저 리서치 (User Research)

### 2.1 목표 사용자 (Target User)
- 해외 리그 한국 선수를 응원하는 **순수 팬층**
- 경기 일정/결과를 빠르게 확인하고 싶은 **라이트 유저**
- 선수 관련 커뮤니티 활동을 원하는 **액티브 유저**
- 고퀄리티 비주얼 콘텐츠를 선호하는 **MZ세대 팬**

#### 4.2.7 Talk 커뮤니티 (세분화) ⭐ 개선

**인스턴트 응원 '탭톡' (Tap-Talk):**

- 경기 집중하면서도 팬들과 실시간 감정 공유 가능

**신고/차단 시스템:**
- 욕설, 비방, 스팸 자동 필터링 (AI)
- 사용자 신고 기능 (3회 누적 시 자동 제재)
- 차단한 사용자 글 숨김
- 관리자 모니터링 대시보드
