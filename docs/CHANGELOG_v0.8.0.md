## 신규 및 변경 사항

### 1. Talk 화면 선수별 필터 기능 추가
- Talk 메뉴에서 해당 선수의 뉴스/콘텐츠만 볼 수 있는 필터 추가
- 선수 클릭 시 해당 선수 관련 게시글만 표시
- 선수별 Talk 필터 UI 상단에 선수 아바타 슬라이더 배치

```
┌─────────────────────────────────────────────────────────────┐
│  Talk                                                        │
├─────────────────────────────────────────────────────────────┤
│  선수 필터: [전체] [이강인✓] [손흥민] [이정후] [안세영]       │
├─────────────────────────────────────────────────────────────┤
│  [전체] [🔥인기] [💬루머] [🎨팬아트]                          │
│                                                             │
│  게시글 목록...                                              │
└─────────────────────────────────────────────────────────────┘
```

### 2. 메인 헤더 테마 선택 기능
- 종목 선택 대신 **테마 선택** 기능으로 변경
- 팀 컬러 기반 테마 빠르게 전환 가능
- 테마 옵션: PSG (Navy), Spurs (White), Bayern (Red), 커스텀 등

```
┌─────────────────────────────────────────────────────────────┐
│  [🎨PSG] [🎨토트넘] [🎨바이에른] [🌙다크] [☀️라이트] →       │
├─────────────────────────────────────────────────────────────┐
│                      메인 콘텐츠                             │
└─────────────────────────────────────────────────────────────┘
```

### 3. 선수 카테고리 관리 - 바둑판 레이아웃

### 4. 오늘의 경기일정 개선

#### 4.1 방영중인 경기 안내 (상단 배치)
- 현재 시각에 방영 중인 경기를 **오늘의 일정 위에 배치**
- LIVE 배지와 함께 강조 표시
- 실시간 스코어 표시

```
┌─────────────────────────────────────────────────────────────┐
│  🔴 LIVE NOW                                                 │
├─────────────────────────────────────────────────────────────┤
│  ⚽ PSG 1 - 0 Marseille | 45' 전반 진행 중                   │
│     이강인 출전 중 | SPOTV                                   │
└─────────────────────────────────────────────────────────────┘
```

#### 4.2 오늘의 경기일정 카드
- "오늘의 일정" → **"오늘의 경기일정"**으로 명칭 변경
- 경기 일정은 **3개만 표시**, 카드 클릭 시 드래그로 더 보기
- 진행 중인 경기는 "진행중" 표시
- 전체 일정 보기 버튼 추가

```
┌─────────────────────────────────────────────────────────────┐
│  📅 오늘의 경기일정                    [전체일정 보기 →]     │
├─────────────────────────────────────────────────────────────┤
│  ┌────────────┐ ┌────────────┐ ┌────────────┐              │
│  │ 05:00      │ │ 09:30      │ │ 14:00      │ ← 드래그    │
│  │ ⚽PSG-OM   │ │ ⚾SF-LAD   │ │ 🏸BWF 결승 │              │
│  │ 진행중🔴   │ │ 예정       │ │ 예정       │              │
│  └────────────┘ └────────────┘ └────────────┘              │
└─────────────────────────────────────────────────────────────┘
```

#### 4.3 리그 이름 및 순위 카드
- 오늘의 경기일정 아래에 **리그 정보 카드** 추가
- 현재 순위 및 리그 현황 표시

```
┌─────────────────────────────────────────────────────────────┐
│  🏆 리그 현황                                                │
├─────────────────────────────────────────────────────────────┤
│  ┌─────────────────────────────────────────────────────┐   │
│  │ Ligue 1 2025-26                                     │   │
│  │ PSG: 2위 (54pts) | 다음 라운드: Round 25            │   │
│  └─────────────────────────────────────────────────────┘   │
│  ┌─────────────────────────────────────────────────────┐   │
│  │ MLB 2026                                            │   │
│  │ SF Giants: 3위 (45-38) | NL West                    │   │
│  └─────────────────────────────────────────────────────┘   │
└─────────────────────────────────────────────────────────────┘
```

### 5. 최근활약 / 최신뉴스 - 선수별 필터링
- 선택된 선수에 맞춰 **최근활약** 데이터 표시
- 선택된 선수에 맞춰 **최신뉴스** 표시
- 선수 전환 시 자동으로 관련 콘텐츠로 업데이트

### 6. 탐색 화면 선수관리 메뉴 추가
- 탐색 화면 상단에 **선수관리 메뉴** 추가
- 종목별 선수 보기 / 전체보기 토글
- 선수 추가/삭제/등록 기능 통합

```
┌─────────────────────────────────────────────────────────────┐
│  탐색                                                        │
├─────────────────────────────────────────────────────────────┤
│  [⚽축구] [⚾야구] [🏸배드민턴] [🏊수영] [⛳골프]              │
├─────────────────────────────────────────────────────────────┤
│  [📋 선수관리]                                               │
│  ├─ 종목별 보기 / 전체보기 토글                              │
│  └─ 선수 추가/삭제 기능                                      │
├─────────────────────────────────────────────────────────────┤
│  선수 목록...                                                │
└─────────────────────────────────────────────────────────────┘
```

## 기술 구현 사항

### 신규 Provider
- `selectedAthleteForTalkProvider`: Talk 화면 선수 필터용
- `themePresetsProvider`: 테마 프리셋 관리
- `liveMatchesProvider`: 실시간 경기 목록
- `leagueStandingsProvider`: 리그 순위 정보

### 신규 Widget
- `AthleteFilterChips`: 선수 필터 칩 위젯
- `ThemeSelector`: 테마 선택 위젯
- `AthleteGridCard`: 바둑판 선수 카드
- `LiveMatchBanner`: 실시간 경기 배너
- `MatchScheduleCarousel`: 경기 일정 캐러셀
- `LeagueStandingCard`: 리그 순위 카드

### 수정된 화면
- `CommunityScreen`: 선수별 필터 추가
- `MainScreen`: 테마 선택으로 변경
- `PlayerManageScreen`: 바둑판 레이아웃
- `HomeScreen`: 경기일정 개선, 리그 카드 추가
- `ExploreScreen`: 선수관리 메뉴 추가
